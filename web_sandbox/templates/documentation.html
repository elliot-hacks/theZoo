🌐 Web Sandbox Documentation

Overview

The theZoo Web Sandbox is a FastAPI-based web interface for browsing and interacting with the theZoo malware database. It provides the same functionality as the original CLI and GUI applications, but through a modern web browser interface.

Features

  • Browse - View all malware samples with filtering options
  • Search - Advanced search by name, type, platform, architecture, language, and tags
  • Details - View comprehensive metadata for each malware sample
  • Download - Download encrypted malware samples with safety warnings
  • REST API - Programmatic access to all functionality
  • Documentation - Built-in access to theZoo documentation

Installation

# Install web sandbox dependencies
cd web_sandbox
pip install -r requirements.txt

# Run the web server
uvicorn main:app --host=127.0.0.1 --port=8000 --reload

Security Considerations

⚠️ IMPORTANT: This web sandbox should ONLY be run in isolated environments.
  • Always bind to 127.0.0.1 (localhost) for security
  • Never expose to public networks
  • Run in isolated virtual machines
  • Use VPN or isolated network if remote access is needed

🔌 REST API Reference

The web sandbox provides a REST API for programmatic access to the malware database. All endpoints return JSON responses.

Method Endpoint Description Parameters
GET /api/stats Get database statistics None
GET /api/malwares List malware samples page, per_page
GET /api/malwares/{id} Get malware details mal_id (path)
GET /api/malwares/search Search malware database q (query string)
POST /api/update-db Update malware database None

API Examples

Get Database Statistics
curl http://localhost:8000/api/stats
Search for Ransomware
curl "http://localhost:8000/api/malwares/search?q=ransomware+win32"
Get Malware Details
curl http://localhost:8000/api/malwares/1
